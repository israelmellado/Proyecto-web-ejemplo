 # 🚀 Proyecto Web Ejemplo


## 1.  📋 Descripción del Proyecto

Este es un **proyecto web de ejemplo** creado para demostrar el uso de **Git y GitHub** en un entorno colaborativo. Incluye una página web moderna con HTML5, CSS3 y diseño responsive.

## 2. **Objetivos del proyecto:**
- ✅ Demostrar flujos de trabajo con Git
- ✅ Manejo de ramas y Pull Requests
- ✅ Resolución de conflictos de merge
- ✅ Buenas prácticas de desarrollo web

##  4.🚀 Cómo usar el proyecto
### 1. Clonar el repositorio
    - ✅ git clone https://github.com/israelmellado/Proyecto-Web-Ejemplo.git
    - ✅ cd Proyecto-Web-Ejemplo
### 2. Abrir en el navegador
 ```
 # Opción 1: Live Server (VS Code extension recomendada)
 # Opción 2: Abrir index.html directamente en el navegador
 open index.html
 # o
 start index.html
 ``` 
### 3. Estructura del proyecto
  ```
  ├── index.html
  ├── css/
  │   └── style.css
  ├── js/
  │   └── main.js
  └── README.md
  ```
